Google Places WP adds the ability to include a Google Map on their Wordpress website that shows nearby places of interest.

== Description ==

Easily add Google Maps with nearby locations setup in categories you can configure to display the map nicely on the frontend. This allows you to show points of interest around a target location. 


== Installation ==

1. Upload the google-places-wp directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the new WP Places link in the WP admin area. 
4. From there, provide your Google Maps API key and click save. After adding your API key with the correct API libraries enabled you can begin configuring the map.
