Google Places WP displays a Google API map with nearby points of interst using a shortcode. 

== Description ==

Google Places WP is a plugin that renders a map using the Google Maps API on the frontend using a simple shortcode after some intial configuration. After providing initial lat/lng coordinates the plugin will allow the user to choose what points of interest nearby the target location should be displayed in the map. 


== Installation ==

1. Upload the google-places-wp directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the new Google Places link in the WP admin area. 
4. From there, provide your Google Maps API key and click save. After saving the key new setting fields will be displayed.
